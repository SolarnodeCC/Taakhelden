Solid/bordered/ghost/danger button for the parent dashboard's calm, business surfaces.

```jsx
<Button variant="primary">Opslaan</Button>
<Button variant="secondary">Annuleren</Button>
<Button variant="danger" size="sm">Verwijderen</Button>
```

Variants: primary (teal fill), secondary (bordered), ghost (text-only), danger (red fill). Sizes: sm/md/lg. `disabled` dims to 60% opacity.
